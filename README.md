# cms50ew_datadump
Download raw stored data from cms50ew pulse oximeter via USB.
Tested with Windows and Python 2.7

## Usage
Edit cms50ew_datadump.py for correct COM port and preferred filename.

Run:

    python cms50ew_datadump.py

Thats it.
